# 🚗 Carros

🔗 **Acesse o projeto:**  
https://carros-ten.vercel.app/

---

## 📌 Sobre o projeto
O **Carros Ten** é uma aplicação web desenvolvida com foco em exibição e interação com informações relacionadas a veículos. O projeto foi criado para praticar conceitos de desenvolvimento front-end, organização de dados e construção de interfaces modernas.

---

## 🚀 Tecnologias utilizadas
- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Funcionalidades
- Exibição de informações sobre carros  
- Interface interativa e responsiva  
- Manipulação dinâmica de dados com JavaScript  
- Navegação simples e intuitiva  

---

## 💡 Objetivo do projeto
Este projeto tem como objetivo:
- Praticar desenvolvimento web  
- Trabalhar manipulação do DOM  
- Melhorar organização de código  
- Criar uma aplicação funcional e visualmente agradável
